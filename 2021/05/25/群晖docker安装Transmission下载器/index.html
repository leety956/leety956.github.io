群晖docker安装Transmission下载器

前言
群晖NAS一般是用download station进行PT下载，但有些客户习惯了Transmission，想安装Transmission，网上好多教程都是用代码安装。

今天就给大家介绍一个纯图形界面安装Transmission的方法。

本篇文章严格核对过，不会出现对着教程操作不成功的问题发生！

相关资料
下面2个链接只是方便大家查阅而已，不用打开直接看后面的操作，是可以正常部署的！

transmission的docker镜像详细了解，可查看：

https://registry.hub.docker.com/u/linuxserver/transmission/

WebUI详细了解，可查看：

https://github.com/ronggang/transmission-web-control

安装docker
没有安装docker套件的，在套件中心搜索docker，点击安装套件即可

准备工作
在正式操作之前，我们需要先做一些准备工作

添加文件夹
打开File station，先在docker目录下建一个“transmission”文件夹

然后在刚刚建的“transmission”文件夹下面建以下4个文件夹

“web”、”watch“、”downloads“、”config“

给文件夹设置Everyone权限
接着我们给docker权限，对着docker的文件夹，点击右键，选择属性，接着按下图操作

=page=

安装镜像
打开docker套件，然后在注册表里，输入transmission，点击搜索，接着选中下图框选的镜像，点击下载

接着会弹出一个对话框，让我们来选择标签，默认的是latest，我们直接点击选择即可（图略）

拉取镜像的时候比较慢，耐心等待即可。

如果实在下载不了，可以切换到国内镜像源，具体方法可以自己百度了解下。后期本文再补充相关教程。

下载好了之后，我们点击映像，能看到大小是90M，接着我们选中它，点击启动。

启动参数设置
参数的设置，大家直接参考本文操作就行

容器名称设置
启动界面里，名称我们可以自己设置，也可以直接默认，接着点击高级设置

下面照着图操作就可以了：

高级设置

卷
点击添加文件夹 –> 选择docker –> 选择transmission –> 选择config

到此transmission就算是搭建好了，但是ui界面还没有，所以我们需要给他上一个汉化的网页界面，方法很简单，接着来操作就可以啦。

WebUI设置操作

把下载的文件解压，然后进入src文件夹！然后进入src文件夹！然后进入src文件夹！

然后把这些文件都拷到群晖里面的docker –> transmission –> web 文件夹里

到此就全部操作完成啦！

访问web界面
访问地址为：IP+9091，例如192.168.1.8:9091。

初次打开的时候，页面的css等静态资源可能还加载不完全，打开网页，多等会刷新就可以了，后面就ok了。
